# Welcome to the Custom Hitster Player
### CHP is a simple Program written in Python which allows you to play the Game Hitster with custom Playlists

## How to play
Sadly because the Spotify Web API does not allow for more than 5 Users on one application this requires some extra Steps and may be hard for the not techy people.

### What you need
- a Spotify account
- a computer

and
either:
- paper
- scissors

or:
- a printer
- the [template](markdown/template.pdf)
